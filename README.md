Welcome to the official repository of **Genius Developers**. This is a high-performance, visually stunning portfolio and digital agency website designed for market dominance and high-conversion user experiences.

---

## ✨ Features

- **🎯 Interactive Animations**: Powered by Framer Motion for a fluid, premium feel.
- **🎨 Modern UI/UX**: Built with Aceternity UI and Tailwind CSS 4 for a state-of-the-art aesthetic.
- **📱 Fully Responsive**: Optimized for every device, from mobile posters to desktop dashboards.
- **🌓 Dark Mode**: Seamless theme switching for optimal viewing.
- **⚡ Performance Optimized**: High-speed loading and SEO-ready architecture.
- **📲 PWA Ready**: Installable application for a native-like experience.

## 🛠️ Tech Stack

- **Core**: Next.js 16 (App Router), React 19, TypeScript
- **Styling**: Tailwind CSS 4, Lucide Icons, React Icons
- **Motion**: Framer Motion
- **Management**: Next Themes, `@ducanh2912/next-pwa`
- **UI Architecture**: Aceternity UI Patterns

## 🚀 Getting Started

To run this project locally:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/The-CyberGenius/Genius-Developers-Main.git
    cd Genius-Developers-Main
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

4.  **Open the application**:
    Navigate to [http://localhost:3000](http://localhost:3000)

---

## 📂 Project Structure

- `app/`: Routing and main pages (Home, Privacy, Terms, variants).
- `components/`:
  - `sections/`: Core website sections (Hero, About, Services, etc.).
  - `ui/`: Shared UI components and custom effects.
- `lib/`: Shared utilities and logic.
- `public/`: Optimized assets and site icons.
